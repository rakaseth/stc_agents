# Fullstack E-commerce App Plugin

Complete fullstack e-commerce solution for small businesses (1K-10K orders/month) with unified backend API and multi-platform frontends.

## Stack

**Backend:**
- Node.js + Express + TypeScript
- Prisma ORM + PostgreSQL
- JWT Authentication
- Stripe/PayPal Integration
- Redis Caching

**Web Frontend:**
- React + TypeScript
- React Query + Zustand
- Tailwind CSS
- React Router

**Mobile:**
- React Native + TypeScript
- React Navigation
- NativeWind (Tailwind for RN)

## Components

### Agents (5)
1. **nodejs-backend-architect** - Express API design
2. **react-web-developer** - React web app
3. **react-native-developer** - Mobile app
4. **api-integration-specialist** - Connect frontend/backend
5. **deployment-specialist** - Deploy full stack

### Commands (4)
1. **backend-setup** - Setup Express API
2. **web-setup** - Setup React web
3. **mobile-setup** - Setup React Native
4. **fullstack-integration** - Wire everything

### Skills (4)
1. **express-api-patterns** - Express best practices
2. **react-native-patterns** - Mobile patterns
3. **api-integration-patterns** - Frontend-backend integration  
4. **fullstack-deployment** - Deployment strategies

## Usage

```bash
# Install plugin
/plugin install fullstack-ecommerce-app

# Setup backend
/fullstack-ecommerce-app:backend-setup

# Setup web frontend
/fullstack-ecommerce-app:web-setup

# Setup mobile app
/fullstack-ecommerce-app:mobile-setup

# Integrate everything
/fullstack-ecommerce-app:fullstack-integration
```

## Features

✅ Complete REST API with Node.js/Express
✅ Prisma database schema for e-commerce
✅ JWT authentication & authorization
✅ Stripe & PayPal payment integration
✅ React web app with responsive design
✅ React Native mobile app (iOS & Android)
✅ Shared TypeScript types across stack
✅ Redis caching for performance
✅ Rate limiting & security middleware
✅ Handles 1K-10K orders/month easily
✅ Deploy to Render/Railway/DigitalOcean
