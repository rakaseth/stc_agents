# Fullstack Integration Command

Integrate React/React Native frontends with Node.js backend API.

## Usage
```bash
/fullstack-ecommerce-app:fullstack-integration
```

## What It Does
1. Setup shared TypeScript types
2. Configure API client with React Query
3. Implement authentication flow
4. Connect all API endpoints
5. Test end-to-end workflows
6. Setup error handling
7. Configure CORS properly

## Output
- Type-safe API integration
- Working authentication
- Complete data flow
- Error handling
