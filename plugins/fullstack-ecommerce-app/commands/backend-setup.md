# Backend Setup Command

Setup complete Node.js/Express backend API with Prisma, authentication, and payment integration.

## Usage
```bash
/fullstack-ecommerce-app:backend-setup
```

## What It Does
1. Initialize Node.js/Express project with TypeScript
2. Setup Prisma with PostgreSQL schema
3. Implement authentication (JWT)
4. Create REST API endpoints (products, cart, orders, users)
5. Integrate Stripe/PayPal
6. Add rate limiting & security middleware
7. Setup Redis caching
8. Create API documentation

## Output
- Complete Express backend
- Prisma schema & migrations
- Authentication system
- Payment integration
- API tests
